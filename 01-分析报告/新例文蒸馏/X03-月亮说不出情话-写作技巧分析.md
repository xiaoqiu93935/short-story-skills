## 一、情绪节奏设计

### 1.1 章节情绪波形

| 章节 | 情绪弧线 | 核心事件 | 情绪值（1-10） |
|------|----------|----------|----------------|
| 第1章 | 平缓→暗涌 | 同学会重逢，沈安煜即将订婚，女主已有家室 | 6 |
| 第2章 | 升温→爆发 | 沈泽挑衅，女主动手，两人在泳池互殴 | 8 |
| 第3章 | 低谷→反转 | 医院抽血求爱、派出所偷听真相 | 7 |
| 第4章 | 蓄势→悬念 | 霍家晚宴入场，女主身份成谜 | 7 |
| 第5章 | 压迫→爽起 | 二楼同框，沈杰被打脸 | 8 |
| 第6章 | 高潮→释然 | 霍衍司官宣"霍夫人"，沈安煜当场崩溃跪求 | 10 |
| 第7章 | 余韵→甜蜜 | 收尾沈杰，教育沈安煜，家庭温情 | 7 |

### 1.2 节奏模式

**核心节奏：压抑-爆发-反转三段式（7:2:1）**

- **70% 压抑蓄势**：女主全程隐忍克制，不解释、不争辩、不动声色，让反派和旁观者不断低估她
- **20% 爆发打脸**：身份揭晓那一刻，暴力碾压，所有人目瞪口呆
- **10% 余韵回味**：以家庭温情收尾，前任跪地哭泣形成鲜明对比

**关键节奏节点（节奏波形标注）：**

```
第1章：开局-平缓湖面
第2章：冲突-第一次肢体爆发（泳池混战）
第3章：回忆-深层揭示（相爱相杀的起源）
第4章：蓄力-身份迷雾（"二楼有人会告诉你"）
第5章：压迫-沈杰持续被打脸（多次巴掌）
第6章：高潮-霍衍司官宣（全场起立）★★★节奏峰值
第7章：收束-家庭温情+前任崩溃余韵
```

### 1.3 情绪钩子设计

| 钩子类型 | 出现位置 | 原文示例 | 效果 |
|----------|----------|----------|------|
| 反差钩 | 第6章 | "惹我尚有路走，惹了我太太，就不要怪我断了各位的路" | 爽感爆发 |

## 二、爽点布局

### 2.1 核心爽点矩阵

| 爽点类型 | 出现频次 | 代表场景 | 效果评级 |
|----------|----------|----------|----------|
| **身份碾压爽** | 高频（4次+） | 霍家晚宴官宣、"保镖鞠躬" | S |
| **火葬场爽** | 高频（3次） | 沈安煜下跪、洗手间崩溃 | S |
| **复仇爽** | 中频（3次） | 砸包、泳池互殴、雨夜收拾沈杰 | A |
| **被低估爽** | 高频（持续） | 全程克制→最后揭晓 | A |
| **护崽爽** | 中频（2次） | 女儿出现+保护场面 | A |

### 2.2 爽点高潮设计（高潮密度分析）

**全文爽点分布：**

```
0%  ─────── 20% ─────── 40% ─────── 60% ─────── 80% ─────── 100%
│           │           │           │           │           │
[悬念积累]  [小爽:砸包]  [泳池互殴]  [晚宴入场]  [官宣爆发]  [温情收尾]
             (沈杰)     (沈安煜)    (身份谜)    (全场跪)    (HE)
```

**高潮密度评估：**
- 第2章小爽 × 1（砸沈泽）
- 第6章大爽 × 3（官宣、沈跪、雨夜制裁）
- 第7章余爽 × 2（缝嘴、女儿温情）
- **整体爽感分布：前缓后爆，爆发力集中于结尾30%**

### 2.3 反派打脸链路

```
沈安煜：采访放狠话 → 被告知是霍夫人 → 官宣现场 → 洗手间下跪 → 全程崩溃
沈  泽：叫"破烂货" → 包砸脸 → 雨夜被踩肋骨 → 高跟鞋贯穿膝盖 → 缝嘴
岳弯弯：示威 → 被推开（"洁癖"） → 替沈挡刀 → 雨中搀扶沈安煜（对照组）
```

## 三、人设反差设计

### 3.1 女主（程舒）人设模型

**外在人设：**
- 曾因复仇入局：出卖公司机密害沈安煜破产
- 母亲被沈安煜害得瘫痪
- 有钱有颜有女儿，已再婚
- 全程克制隐忍，有洁癖

**内在人设：**
- 手段狠辣（泳池动刀、雨夜收拾沈杰）
- 护崽心切（听到沈杰骂女儿，直接踩断膝盖）
- 心态平稳（"我心态好了，不是手段没了"）
- 被霍衍司彻底治愈，原则在他面前不攻自破

**反差维度：**
| 维度 | 表层 | 深层 |
|------|------|------|
| 社交 | 礼貌克制（"你好"） | 暗藏杀机 |
| 情绪 | 平和哄女儿 | 瞬间爆发 |
| 身份 | 低调参加聚会 | 霍家女主人 |
| 手段 | 表面不计较 | 手段用过之人 |

### 3.2 男二（霍衍司）人设模型

**反差设计：**
- 商场上：冷面威严，"惹我太太断你后路"
- 家庭里：粘人撒娇，靠在老婆怀里
- 手段上：亲手策划"一千只蚂蚁放骨头里"的酷刑
- 情感上：知道老婆受过伤，全力治愈

**功能定位：** 救赎者/工具人丈夫，用于对比前任的毁灭性爱情

### 3.3 反派（沈安煜）人设模型

**6年互斗背景设定（恨的根源）：**
- 求婚当天 → 女主卖红头文件 → 沈破产
- 答应结婚那天 → 沈拔氧气罐 → 女主妈瘫痪
- 出车祸 → 两人在ICU互相伤害

**重逢后心理变化路径：**
```
见面：放狠话施压（想看她崩溃）
中期：被无视激怒（"你还是不装的样子更让我顺眼"）
后期：发现她已婚有女（彻底崩溃）
结尾：跪求离婚，觊觎看着女儿
```

## 五、开篇必杀技

### 5.1 开篇第一句分析

**原文：**
> 我跟沈安煜明争暗斗了六年。

**分析：**
- 直接交代核心矛盾（明争暗斗 + 六年）
- 两个名字并列，无主语，暗示关系对等
- "明争暗斗"四字涵盖所有后文
- 无废话，零铺垫，直接入戏

**评分：S**

### 5.2 前三段功能设计

| 段落 | 内容 | 功能 |
|------|------|------|
| 第1句 | 明争暗斗六年 | 核心矛盾确立 |
| 第2句 | 互相伤害细节 | 展示仇恨深度 |
| 第3句 | 众人预期 | 反差铺垫（实际已各自幸福） |

### 5.3 开篇类型判断

**属于：悬念钩开篇**
- 制造认知冲突："互相折磨一辈子" vs 实际各自幸福
- 读者期待管理：期待看到和解还是继续互撕？
- 快速建立双重悬念：过去为什么斗？现在如何收场？

## 六、光幕写法（场景切换/叙事技巧）

### 6.1 场景切换技法

**主要场景流：**
```
同学会 → 室外草坪（打电话） → 聚会室内 → 泳池 → 医院 → 派出所 → 霍家晚宴 → 雨夜室外 → 家中
```

**关键转场技巧：**

| 转场位置 | 前场景 | 后场景 | 技巧 |
|----------|--------|--------|------|
| 第3章开头 | 泳池互殴 | 医院 | 事件结果承接 |
| 第3章中 | 派出所 | 霍衍司接人 | 声音转场（议论声） |
| 第4章开头 | 派出所 | 车内 | 对话承接 |
| 第6章 | 晚宴 | 洗手间外 | 地理移动 |
| 第7章 | 雨夜 | 家中 | 时间+情感承接 |

### 6.2 叙事视角

- **主视角**：女主第一人称（内心独白丰富）
- **信息控制**：女主知道的事情 > 读者知道 > 其他角色知道
- **关键信息延迟**：女主已婚有女这件事，在第1章就埋了戒指钩子，但到第6章才官宣

### 6.3 多线叙事

| 线索 | 状态 | 交织方式 |
|------|------|----------|
| 女主vs沈安煜 | 仇恨线（明） | 全程对抗 |
| 女主vs霍衍司 | 恩爱线（暗） | 闪回补充，结尾显山露水 |
| 沈杰vs女主 | 炮灰线 | 多次打脸递进 |
| 岳弯弯vs女主 | 对照线 | 对比"失败者"与"成功者" |

## 七、类型归类（22主类型体系）

### 7.1 一级类型：豪门甜宠

**核心特征：** 女主机缘巧合/蓄意接近顶级权贵，被宠爱逆袭

**本文对应要素：**
- 霍衍司是顶级豪门掌权者
- "霍夫人"身份碾压全场
- 霍衍司护短：发布会发言、晚宴官宣、雨夜动手
- 闪婚闪孕+幸福日常

### 7.2 二级类型：追妻火葬场

**核心特征：** 前任幡然醒悟/追悔莫及，女主已然幸福

**本文对应要素：**
- 沈安煜从"弄残她妈"到"跪求离婚"
- 洗手间崩溃："为什么连恨都不恨我了"
- 看到女儿后彻底破碎："对不起……对不起……"
- 火葬场强度：★★★★★（跪地+落泪+觊觎女儿）

### 7.3 三级类型：身份碾压

**核心特征：** 女主隐藏身份，反派当面挑衅，最后揭晓身份碾压

**本文对应要素：**
- 第1章：沈杰叫她"破烂货"
- 第4章：沈安煜猜测她"钓大鱼"
- 第5章：沈杰让她"滚下去"
- 第6章：霍衍司官宣，全场起立，沈安煜眼角泛红

### 7.4 四级类型：双向复仇

**核心特征：** 男女主互相伤害，爱恨交织

**本文对应要素：**
- 女主卖文件 → 沈让女主妈瘫痪
- 求婚当天卖红头文件
- ICU互砍互抠互相拔管
- 泳池动刀

### 7.5 五级类型：救赎治愈

**核心特征：** 新男主治愈女主旧伤

**本文对应要素：**
- 霍衍司填平负面情绪
- 治好女主妈的腿
- "手段用来保护爱人，而不是加注彼此"
- 女儿降生，家庭幸福

### 7.6 类型组合公式

```
本文类型 = 豪门甜宠（主）+ 追妻火葬场（核）+ 身份碾压（爽）+ 双向复仇（虐）+ 救赎治愈（暖）
```

## 八、写作模板提炼

### 8.1 全文结构模板

```
【开篇钩子】
第一句：核心矛盾（明争暗斗X年）
↓
【第一幕：重逢】
- 暗示女主已有新生活（戒指/女儿电话）
- 反派嚣张挑衅
- 女主克制隐忍（被误会、被叫破烂货）
↓
【第二幕：冲突升级】
- 小冲突：砸包/泳池互殴
- 中冲突：女主展现真实力
- 插叙：揭示相爱相杀的过去
↓
【第三幕：身份打脸】
- 晚宴入场（保镖行礼铺垫）
- 沈杰持续打脸
- 霍衍司官宣（全场高潮）
↓
【第四幕：火葬场+收尾】
- 沈安煜崩溃跪地
- 雨夜制裁沈杰
- 家庭温情HE
```

### 8.2 核心人设模板

**女主公式：**
```
外表：美丽/克制/有洁癖/不解释
背景：有伤/有仇/有过去
能力：能让敌人破产一次，就能第二次
情感：被现任治愈/护崽/原则遇到真爱不攻自破
爽点：被低估→身份揭晓→全场震惊
```

**火葬场男主公式：**
```
曾经：毁灭性爱情（互相伤害）
重逢：想看她崩溃/放狠话
中期：被无视激怒
结局：发现她已婚有娃→彻底崩溃
高潮：跪地求复合→被无视→绝望
```

**救赎男主公式：**
```
商场：冷面阎王/只手遮天
家庭：撒娇粘人/宠妻无度
手段：比女主还狠（"蚂蚁放骨头"）
金句：手段是用来保护爱人
```

### 8.3 场景打脸模板

**多级打脸递进结构（适用于任何打脸场景）：**

```
Level 1（言语挑衅）：
反派："破烂货，你算老几？"
女主：不回应，起身

Level 2（小动作）：
反派：以为她要撕脸，按住她
女主：说"去洗手间"

Level 3（继续低估）：
旁观者议论："放不下""两年了还惦记"

Level 4（身份揭晓）：
保镖鞠躬："夫人"
女主：淡定入座

Level 5（权力碾压）：
霍衍司官宣："惹我太太，断各位的路"

Level 6（彻底崩溃）：
反派：眼角泛红 / 跪地 / 落泪
```

### 8.4 金句模板库

| 类型 | 句式 | 例子 |

| 复仇宣言 | "我能让他___一次，就能让他___第二次" | "我能让他破产一次，就能让他破产第二次" |
