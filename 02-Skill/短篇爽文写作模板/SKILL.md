## 核心工作流

### Step 1: 例文输入

接收用户的例文输入，支持以下方式：
- 直接提供文件路径（txt/md/docx等）
- 直接粘贴文本
- 提供文件夹路径（批量分析）

**确认信息**：
1. 分析深度：快速概览 / 标准分析 / 深度蒸馏
2. 是否需要生成写作模板
3. 是否需要与已有模板库对比

### Step 2: 八维度分析

对每篇例文执行以下8个维度的分析：

---

#### 维度1：叙事结构

**提取重点**：
- 结构类型（五段式/三幕式/双线叙事/片段式）
- 篇幅分配比例（铺垫/发展/转折/高潮/结局各占多少）
- 节奏控制（钩子密度、信息释放速率）
- POV策略（第一人称/第三人称/多视角切换）

**输出格式**：
```yaml
结构类型: [类型]
篇幅分配: 铺垫X% → 发展Y% → 转折Z% → 高潮W% → 结局V%
钩子密度: 每N字一个钩子
POV策略: [描述]
章节总数: N章
```

---

#### 维度2：爽点布局

**提取重点**：
- 爽点层次（表/中/深/终极）
- 爽点类型分布（打脸/掉马/团宠/护崽/逆袭/复仇/其他）
- 爽点密度（每N字一个爽点）
- 铺垫-爆发间隔（铺垫多长后爆发）
- 爽点实现公式（可复用的结构）

**输出格式**：
```yaml
爽点层次: X层
爽点类型: 打脸X次/掉马X次/团宠X次/逆袭X次/其他X次
爽点密度: 每N字一个爽点
平均铺垫间隔: N字
核心爽点公式: [公式]
```

---

#### 维度3：人物设定

**提取重点**：
- 主角表里两层设定（表面vs真实身份/动机）
- 人物弧光轨迹（起点→转折→终点）
- 反派暴露节奏（渐进式/突然式/自爆式）
- 反派智商控制方式（欲望降智/信息差/自大）
- 配角功能分配

**输出格式**：
```yaml
主角:
  表面: [描述]
  真实: [描述]
  弧光: [起点] → [转折] → [终点]
反派:
  暴露方式: [方式]
  暴露层数: N层
  降智方式: [欲望/信息差/自大]
配角:
  - [角色名]: [功能]
```

---

#### 维度4：悬念与反转

**提取重点**：
- 反转数量和类型（身份反转/情节反转/认知反转/世界观反转）
- 悬念埋设-揭示间隔
- 信息差设计（读者vs主角vs反派的信息不对称）
- 伏笔-回收表

**输出格式**：
```yaml
反转:
  数量: N个
  类型: 身份反转X次/情节反转X次/认知反转X次/世界观反转X次
  最大悬念间隔: N章
信息差模式: [描述]
伏笔回收: [N条伏笔，回收N条]
```

---

#### 维度5：世界观构建

**提取重点**：
- 核心设定（一句话概括）
- 世界观揭示节奏（哪些信息在什么时候透露）
- 设定与剧情的绑定方式
- 设定补全时机

**输出格式**：
```yaml
核心设定: [一句话]
揭示时间表:
  - 第X章: 揭示[信息]
  - 第Y章: 揭示[信息]
设定-剧情绑定: [描述]
设定补全时机: [故事中/结尾/未补全]
```

---

#### 维度6：情感操控

**提取重点**：
- 情感曲线（读者情绪随章节的变化）
- 情感操控手法（共情建立/愤怒积累/期待管理/释放节奏）
- 关键情感转折点

**输出格式**：
```yaml
情感曲线: [描述波形]
操控手法:
  - 共情建立: [方式]
  - 愤怒积累: [方式]
  - 期待管理: [方式]
  - 释放节奏: [方式]
关键转折点: [列举]
```

---

#### 维度7：语言风格

**提取重点**：
- 叙事节奏（短句/长句比例）
- 感官描写密度与类型
- 对话vs叙述比例
- 情感表达方式（直白/含蓄/反差）
- 标志性句式/用词

**输出格式**：
```yaml
叙事节奏: [短句为主/长短交替/长句为主]
感官描写: [视觉为主/多感官/极简]
对话/叙述比: X:Y
情感表达: [方式]
标志性句式: [列举]
```

---

#### 维度8：商业属性

**提取重点**：
- 标题公式（标题的结构分析）
- 开篇吸引力（前3句话的钩子）
- 读者留存策略（每章结尾的钩子类型）
- 爽点密度是否达标

**输出格式**：
```yaml
标题公式: [分析]
开篇钩子: [类型+效果]
章尾钩子策略: [描述]
爽点达标: [是/否，理由]
```

---

### Step 3: 技巧提炼

从8维度分析结果中，提炼可复用的写作技巧。每个技巧需通过**三重验证**：

| 验证维度 | 判断标准 | 通过等级 |
|---------|---------|---------|
| **跨例文复现** | 在≥2篇不同例文中出现？ | 通用技巧 |
| **生成力** | 能指导创作新故事？ | 可操作技巧 |
| **排他性** | 不是所有故事都这样写？ | 独特技巧 |

- 三重通过 → **S级核心技巧**（写入模板库，强制遵循）
- 通过2重 → **A级参考技巧**（写入模板库，推荐遵循）
- 通过1重 → **B级备选技巧**（标注适用场景）
- 0重 → 丢弃

**输出格式**：
```
技巧名称: [名称]
级别: S/A/B
来源: [例文名] + [具体章节]
验证: 跨例文[是/否] | 生成力[是/否] | 排他性[是/否]
公式: [可复用的结构化公式]
示例: [原文摘录]
适用场景: [描述]
局限: [描述]
```

---

### Step 4: 模板生成

基于分析结果，生成以下类型的写作模板：

#### 4.1 叙事结构模板
```
开篇（X%）：[具体操作]
发展（Y%）：[具体操作]
转折（Z%）：[具体操作]
高潮（W%）：[具体操作]
结局（V%）：[具体操作]
```

#### 4.2 爽点布局模板
```
爽点1（第X章）：[类型] + [铺垫方式] + [爆发方式]
爽点2（第Y章）：...
...
终极爽点（第Z章）：[类型] + [铺垫方式] + [爆发方式]
```

#### 4.3 人物设定模板
```
主角：表面[...] → 真实[...] → 弧光[...]
反派：暴露方式[...] → 降智方式[...]
配角：[角色1-功能]、[角色2-功能]...
```

#### 4.4 章节功能表
| 章节 | 功能 | 爽点类型 | 技巧标记 | 钩子类型 |
|------|------|----------|----------|----------|

#### 4.5 质量自检清单
根据本篇例文提炼的特定检查项。

---

### Step 5: 模板库更新

将本次分析结果整合到模板库中：

1. **新技巧入库**：通过三重验证的技巧写入模板库
2. **已有技巧强化**：如果新例文支持已有技巧，补充案例
3. **矛盾标记**：如果新例文与已有技巧矛盾，标注分歧
4. **类型扩展**：如果是新类型，创建新的类型模板
5. **版本更新**：记录更新内容

---

## 批量蒸馏流程

当用户提供多个例文时：

### Phase 1: 独立分析
- 对每篇例文独立执行Step 2-4
- 每篇产出独立的分析报告和模板

### Phase 2: 交叉对比
- 找出所有例文的共通技巧
- 找出每篇的独特技巧
- 按三重验证法分级

### Phase 3: 整合输出
- 通用模板（所有例文共通的技巧）
- 分类模板（按类型区分的技巧）
- 独特技巧清单（每篇的独特之处）

### Phase 4: 模板库更新
- S级技巧写入通用模板
- A级技巧写入分类模板
- B级技巧记录在案

## 已有模板库

### 通用技巧库（基于47+篇例文）

> 以下技巧来自「短篇爽文写作术」v1.5 的47篇例文蒸馏，已通过三重验证。

#### 核心心智模型（11个）

1. **情绪过山车定理**：核心不是虐或爽，而是情绪起伏的频率和幅度
2. **背叛链定理**：最虐的是最信任的人背叛，背叛链越深打脸越爽
3. **观众视角特权**：让读者比反派更早知道真相
4. **消失的证人**：女主消失后反派集团内部撕裂
5. **独立收尾法则**：最终胜利不是复仇成功，而是不再需要任何人
6. **身份重选定理**：重新选择人生岔路，亲眼见证两种结局
7. **反手规则借用**：不正面硬刚，借用对方规则反杀
8. **恶役诱导定理**：先让读者误判反派，真相揭露时冲击力翻倍
9. **不阻拦定理**：知道反派会作死，选择不阻拦让其自投罗网

### 新增类型：恐怖猎杀型（基于「善」女村）

**代表作**：「善」女村
**核心矛盾**：主角表面是猎物，实际是猎手→反派自投罗网→集体猎杀→身份重生

```
【人设】
- 主角：表面弱势（需要被拯救），实际是布局者
- 反派：渐进式暴露（表面好人→贪财→色欲→罪犯→猎物）
- 配角集体：整个社区都是猎手（配合布局）

【虐点节奏】
第1-2章：世界观建立（神秘设定+紧迫期限）
第3-4章：反派暴露真面目（贪财/好色/作恶）
第5-6章：主角表面"无奈妥协"（实际布局）
第7-8章：增加难度（规则阻碍/时间紧迫）
第9-11章：反派自投罗网（色欲驱使/贪婪驱动）
第12-13章：仪式准备（双重信息差）
第14-15章：反派发现真相（POV切换，恐怖升级）
第16-17章：抓捕反派（猎物无力化）
第18-20章：身份全部揭示（多层反转）
第21-22章：世界观闭环（设定补全+余韵）

【核心公式】：
  诱饵→陷阱→猎杀 三段式
  
  诱饵阶段：主角伪装弱势 + 展示利益（美女/财富）
  陷阱阶段：反派因欲望降低智商 + 配合者诱惑
  猎杀阶段：集体行动 + 反派失去反抗能力 + 身份重生

【独特技巧】：
  1. 双重叙事层：读者以为在读X类型，实际是Y类型
  2. 反派自投罗网：不是主角反击，是反派自己走进陷阱
  3. 集体智慧：不是一人开挂，是整个社区配合
  4. 世界观颠覆：终极爽点不是复仇成功，而是读者发现世界观完全不同于预期
```

### 新发现S级技巧（7个）
