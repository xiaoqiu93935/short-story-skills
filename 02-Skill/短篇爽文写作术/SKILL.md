## 核心心智模型（9个）

### 模型1：情绪过山车定理
短篇爽文的核心不是"虐"或"爽"，而是**情绪起伏的频率和幅度**。
```
绝望 → 压抑 → 最低谷 → 假象 → 观众视角 → 爽感释放 → 独立收尾
```
每章必须有情绪波动，不能连续3章纯压抑。

### 模型2：背叛链定理
最虐的不是敌人伤害你，而是**你最信任的人背叛你**。
| 背叛者 | 背叛程度 | 爽感潜力 |
|--------|---------|---------|
| 丈夫/妻子 | 婚姻背叛 | ★★★★★ |
| 父亲/母亲 | 血缘背叛 | ★★★★★ |
| 闺蜜/兄弟 | 友情背叛 | ★★★★☆ |

### 模型3：观众视角特权
让读者**比反派更早知道真相**，制造信息差优越感。
- 女主视角揭露真相（第1章炸弹）
- 反派视角继续蒙在鼓里（第2-5章）
- 读者享受"我知道你不知道"的快感

### 模型4：消失的证人
女主"死"或"消失"后，**反派集团开始内部撕裂**。消失是反击的起点，不是终点。

### 模型5：独立收尾法则
女主最终的胜利不是复仇成功，而是**不再需要任何人**。不原谅型收尾爽感最高（★★★★★）。

### 模型6：身份重选定理
让主角**重新选择人生岔路**，并亲眼见证两种选择的截然不同结局。爽感来源：观众已知A选项的悲剧结局，主角主动选B形成对比张力。

### 模型7：反手规则借用
被欺压时**不正面硬刚，而是借用对方规则反杀**。四步：被规则压制 → 表面顺从 → 反向利用 → 规则制定者被规则反噬。

### 模型8：恶役诱导定理
让读者**先以为某人是反派**，再揭露真相，让读者意识到自己被骗了。经典案例：《女儿照顾了我十年》——母亲看似偏心，实则故意引导凶手自曝。

### 模型9：不阻拦定理
女主知道反派会作死，**选择不阻拦**，让反派自己走进陷阱。爽感：比亲手惩罚更有快感——对方是自己找死。

## S级核心技巧（7个）

> 通过三重验证（跨例文复现+生成力+排他性），写入通用技巧库强制遵循。

### S级技巧1：亡灵视角叙事
**公式**：死亡→旁观→无法交流→释怀
**模板**：开篇主角死亡 → 灵魂视角双线叙事 → 受害者vs加害者同步对比 → 真相揭露 → 释怀/复仇

### S级技巧2：预告式钩子+金句锚定
**公式**：预告→金句→回扣
**模板**：开篇预告结局 → 每3章一句可传播金句 → 高潮金句再现冲击翻倍 → 收尾金句闭环

### S级技巧3：儿童有限视角
**公式**：儿童感知→读者脑补→冲击翻倍
**模板**：只写孩子感知的，不解释成人世界，让读者通过孩子眼睛"看见"最残酷的真相

### S级技巧4：律师解谜式叙事
**公式**：接案→调查→庭审→身世反转
**模板**：神秘委托人 → 3层证据揭秘 → 庭审专业打脸 → 身世身份炸弹

### S级技巧5：限知视角递进式悬疑
**公式**：限知+证据递进+每集反转
**模板**：主角记忆不完整 → 新证据推翻前论 → 每3章反转 → 终极真相颠覆认知

### S级技巧6：旁观者介入穿书
**公式**：结局后→旁观者身份→搞钱→攻略反派
**模板**：原著大结局后穿书 → 不介入主线 → 先经济独立 → 慢慢攻略反派

### S级技巧7：双线时间同步对比
**公式**：受害+享受同时→愤怒拉满
**模板**：同一章切换POV → 明确时间锚点 → 受害者越惨+加害者越爽 → 每3章用一次

## 表达DNA

### 句式偏好
- **短句为主**：情绪激动时用短句，制造冲击力
- **重复强调**：关键台词重复出现，强化记忆
- **对比句式**：「曾经...现在...」制造落差

### 情绪词库
| 情绪 | 可用语 |
|------|--------|
| 绝望 | 麻木、血泪、崩塌、窒息 |
| 压抑 | 地下室、囚禁、无声、冰冷 |
| 反击 | 点燃、转身、离开、不回头 |
| 复仇 | 直播、监控、揭露、挫骨扬灰 |
| 独立 | 不原谅、不回头、重新开始 |

### 确定性表达
- 女主内心独白：**斩钉截铁**
- 反派辩解：**虚伪苍白**
- 旁观者评价：**一针见血**

## 套路公式速查

### 标题公式
```
[X]未尽，[Y]无计
你[做某事]，我[反手反击]
[事件]？[态度]，[反手行动]：[结果]
[事件名]后，我[做某事]
查出[疑似疾病]后，我[做某事]
```

### 开篇公式
```
[场景]，[身份A]对[身份B]说："[炸弹内容]"
炸弹类型：表白/背叛/身世/死亡
```

### 情绪转折词
```
绝望→麻木→压抑→窒息
反击→离开→消失→假死
观众视角→证据→打脸→复仇
独立→新生→不原谅→自由
```
