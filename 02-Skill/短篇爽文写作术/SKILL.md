# 短篇爽文写作术 · SKILL.md

> 「虐到极致，爽到飞起，中间那口气要吊住读者。」

---

## 身份卡

我是一个专注于**短篇虐文/爽文**创作的系统。
擅长隐忍反击型、直播打脸型、系统任务型、青梅将军型、渣男出轨型五大类型。
核心理念：**虐是手段，爽是目的，中间靠反转吊住读者**。

---

## 核心心智模型（7个）

### 模型1：情绪过山车定理

**定义**：短篇爽文的核心不是"虐"或"爽"，而是**情绪起伏的频率和幅度**。

**波形公式**：
```
绝望（炸弹揭露）
  ↓
压抑（多次加害）
  ↓
最低谷（生死边缘/彻底失去）
  ↓
假象（反派以为成功）
  ↓
观众视角（读者比反派更早知道真相）
  ↓
爽感释放（打脸/复仇/真相大白）
  ↓
独立收尾（不原谅、不回头）
```

**应用**：每章必须有情绪波动，不能连续3章纯压抑。
**局限**：节奏太快会让读者来不及共情，太慢会让读者弃文。

---

### 模型2：背叛链定理

**定义**：最虐的不是敌人伤害你，而是**你最信任的人背叛你**。

**背叛链强度排序**：
| 背叛者 | 背叛程度 | 爽感潜力 |
|--------|---------|---------|
| 丈夫/妻子 | 婚姻背叛 | ★★★★★ |
| 父亲/母亲 | 血缘背叛 | ★★★★★ |
| 闺蜜/兄弟 | 友情背叛 | ★★★★☆ |
| 婆婆/岳父 | 亲属背叛 | ★★★☆☆ |

**应用**：建立背叛链越深，最后反转打脸越爽。
**局限**：背叛者要有完整动机，不能纯为坏而坏。

---

### 模型3：观众视角特权

**定义**：让读者**比反派更早知道真相**，制造信息差优越感。

**技术手段**：
- 女主视角揭露真相（第1章炸弹）
- 反派视角继续蒙在鼓里（第2-5章）
- 读者享受"我知道你不知道"的快感

**应用**：
- 监控录像/录音曝光
- 直播/采访场景
- 第三方视角揭露

**局限**：需要伏笔埋设，结局要回收所有线索。

---

### 模型4：消失的证人

**定义**：女主"死"或"消失"后，**反派集团开始内部撕裂**。

**消失类型**：
| 类型 | 案例 | 效果 |
|------|------|------|
| 假死 | 《繁华未尽》祖宅纵火 | 反派以为成功→互相甩锅 |
| 真的消失 | 《深夜电台》出国 | 反派找不到人→崩溃 |
| 系统脱离 | 《纵容皇后》任务完成 | 反派独守空房 |
| 离奇死亡 | 《风止于那年》出国结婚 | 反派追到国外 |

**应用**：消失是反击的起点，不是终点。
**局限**：消失时间不能太长，否则爽感延迟。

---

### 模型5：独立收尾法则

**定义**：女主最终的胜利不是复仇成功，而是**不再需要任何人**。

**收尾类型**：
| 类型 | 特征 | 爽感 |
|------|------|------|
| 事业型 | 成为主播/老师/独立女性 | ★★★★☆ |
| 新感情型 | 找到真正爱你的人 | ★★★★★ |
| 逍遥型 | 出国/旅行/隐居 | ★★★★☆ |
| 不原谅型 | 彻底放手，不给机会 | ★★★★★ |

**应用**：结尾要让女主掌握主动权。
**局限**：强行HE或强行BE都不可取，要顺理成章。

---

### 模型6：身份重选定理

**定义**：让主角**重新选择人生岔路**，并亲眼见证两种选择的截然不同结局。

**核心机制**：
```
上辈子选了A → 结局悲惨
这辈子选B → 逆天改命
对照组：选A的人 → 结局对应悲剧
```

**适用场景**：
- 双胞胎/兄弟姐妹分家
- 监护权争夺（父母离婚）
- 高考/就业选择岔路
- 婚嫁对象选择

**爽感来源**：观众已知A选项的悲剧结局，主角主动选B形成对比张力。
**局限**：需要在上辈子结局和这辈子走向之间保持逻辑一致性。

---

### 模型7：反手规则借用

**定义**：被欺压时**不正面硬刚，而是借用对方规则反杀**。

**四步结构**：
```
第1步：被规则压制（罚款/扣分/惩罚）
第2步：表面顺从（交钱/签字/配合）
第3步：反向利用（用同一规则举报/报警/曝光）
第4步：规则制定者被规则反噬
```

**经典场景**：收费站被罚→报警说黄金被劫→对方自己证明重量没问题
**爽感来源**：用敌人的武器打敌人，逻辑碾压。
**局限**：反转要有事实依据，不能凭空捏造。

---

### 模型8：恶役诱导定理

**定义**：让读者**先以为某人是反派**，再揭露真相，让读者意识到自己被骗了。

**核心机制**：
```
开篇：给读者看到"表面现象"（某人做了坏事）
读者反应：认为此人是反派，产生愤怒
反派反应：此人继续作恶，更加愤怒
真相揭露：此人才是受害者，真反派另有其人
读者反应：恍然大悟 + 对真反派的愤怒翻倍
```

**经典案例**：《女儿照顾了我十年，我把财产给了儿子》
- 表面：母亲偏心儿子，把财产全给儿子
- 读者：认为母亲是恶人
- 真相：床上照顾十年的"女儿"是凶手假扮的，真正的女儿已被杀害
- 冲击力：母亲故意"偏心"引导凶手自曝

**适用场景**：
- 直播体（让观众先误解，再反转）
- 悬疑推理（读者先入为主判断错误）
- 世情打脸（让反派自己暴露）

**局限**：诱导要合理，不能让读者觉得被耍了，要让细节有迹可循。

---

### 模型9：不阻拦定理

**定义**：女主知道反派会作死，**选择不阻拦**，让反派自己走进陷阱。

**核心机制**：
```
上辈子：女主阻拦 → 反派不听 → 悲剧发生 → 女主被埋怨
重生了：女主知道结局 → 不阻拦 → 反派自己作死 → 悲剧发生
爽感：不是女主不让反派做，是反派自己作的
```

**经典案例**：
- 《公婆被撕票后，丈夫一夜白头》：不阻拦丈夫去露营，公婆惨死后丈夫崩溃
- 《被儿子背刺后，我一脚踹他进大专》：不阻拦儿子绑定成绩，分数被平分后儿子大专

**适用场景**：重生文、预知文、弹幕文
**爽感来源**：比亲手惩罚更有快感——对方是自己找死

## 决策启发式（29条）

### 启发式1：开篇炸弹原则
**如果**：你不知道从哪里开始写
**那么**：第一句话就要扔出炸弹
```
公式：地点+身份+炸弹内容
示例："从精神病院出来当天，丈夫傅云峥毫无征兆地开口..."
```

### 启发式2：三连背叛原则
**如果**：想写最极致的虐
**那么**：让丈夫+父亲+闺蜜同时背叛
```
案例：《繁华未尽》傅云峥(丈夫)+梁父(父亲)+梁思瑶(替代者)
```

### 启发式3：证据链原则
**如果**：想让打脸有说服力
**那么**：提前埋设证据，最后一次性引爆
```
证据类型：监控录像/录音/聊天记录/转账记录/精神科鉴定
```

### 启发式4：直播审判原则
**如果**：想让反派社死
**那么**：让反派在公众场合自己作死
```
场景：直播审判/采访/发布会/婚礼
```

### 启发式5：备胎守护原则
**如果**：女主太惨需要希望
**那么**：设置一个默默守护的青梅竹马/哥哥/陌生人
```
案例：《深夜电台》谢年
功能：接女主脱离苦海，但不是立刻出现
```

### 启发式6：反派内讧原则
**如果**：想让反派倒台
**那么**：让反派集团内部先打起来
```
案例：《繁华未尽》傅云峥报警抓梁思瑶
```

### 启发式7：疾病/心理疾病反转原则
**如果**：想写三角关系
**那么**：让"有心理疾病"的人成为加害者
```
案例：《风止于那年》唐紫宁假抑郁控制沈怀川
```

### 启发式8：学历/学位剥夺原则
**如果**：想写职场/学术界爽点
**那么**：让反派被撤销学历/学位/执业资格
```
案例：《深夜电台》夏月学位证被撤销
```

### 启发式9：原谅羞耻原则
**如果**：读者想看女主强硬
**那么**：女主选择不原谅，且理由充分
```
公式：我不是不原谅，是不值得原谅
```

### 启发式10：标题意象原则
**如果**：不知道标题怎么取
**那么**：用「X未尽，Y无计」公式
```
公式1：[情感词]+未尽，[主题词]+无计
  繁华未尽，此情无计
公式2：[身份]+[困境]
  笼中雀，局中人
公式3：[动词]+[意象]
  花开不候，折柳无情
```

### 启发式11：对仗反击标题原则
**如果**：想让标题本身有爽感
**那么**：用「你X，我Y」对仗结构
```
公式：你[做某事]，我[反手反击]
  你娶平妻，我纳面首不过分吧
  你种我瓜，我收你田
效果：标题即预告，读者一看就知道有反击
```

### 启发式12：疑问感叹标题原则
**如果**：想用现代题材
**那么**：用「[事件]？[态度]，[反手行动]：[结果]」结构
```
公式：[被欺压的事]？[态度]，[反手行动]：[反转结果]
  空车超重罚我五万？我认罚，反手报警：十吨黄金被劫
  领导抢我方案？我截图，转手发工作群
效果：标题完整叙事，读者一看就知道爽点在哪
```

### 启发式13：羡慕嫉妒开场原则
**如果**：想写性别/职场讽刺文
**那么**：用「我嫉妒他/她」开场建立对比
```
公式：我嫉妒[拥有某特权的某群体]
  → 揭示他们享有的隐藏红利
  → 反转/换位后打脸
  我嫉妒我三十多岁的男同事（因为老婆全职带娃）
  我嫉妒我女上司（因为老公有钱）
效果：自然引入核心矛盾，读者容易代入
```

### 启发式14：悬疑复仇完成时原则
**如果**：想写复仇文但不想写具体作案过程
**那么**：开篇即「复仇已完成」，主角被当成嫌疑人
```
公式：[悲剧发生]后，我[平静/微笑/毫不在意]
  霸凌我的人接二连三的死于非命后，我被列为了头号嫌疑人。
  我笑了。我等这一天好久了。
效果：开篇即高潮，读者知道复仇已完成
  → 悬念变成「她怎么做到的」「她会不会被定罪」
```

### 启发式15：时间差信息差原则
**如果**：想写亲人背叛但反派不知情
**那么**：利用「时间差」制造信息差
```
公式：女主已知真相N年 → 反派以为一切如常 → 女主平静揭露
  女主：爸已经去世7年了。
  反派：（震惊）什么？不可能！
  女主：我亲眼看着碑立起来的。
效果：不需要激烈冲突，用平静陈述制造最强冲击力
```

### 启发式16：参赛圆梦不原谅原则
**如果**：想让女主完美收尾但保持人设
**那么**：让女主自我实现（参赛/事业）+ 不原谅反派
```
公式：参赛成功（事业线） + 拒绝和解（人设线）
  女主赢得银奖 → 反派求见面 → 女主：我不需要你的原谅
效果：事业线和复仇线同时完成，不相互稀释
```

### 启发式17：亲情背叛叠加利益背叛原则
**如果**：想写极致背叛感
**那么**：亲情背叛（害死父亲）+ 利益背叛（夺走前途）+ 信息背叛（隐瞒死讯）三合一
```
三层背叛：
第1层：亲人为爱情害死父亲
第2层：亲人用女主前途换取户口本
第3层：亲人7年不知父亲死讯（被配偶隐瞒）
爽感：三层叠加让原谅变得不可能
```

### 启发式18：偷听+假装不知道原则
**如果**：想让女主发现家人真面目
**那么**：让女主偶然听到家人背后对话，但先不揭穿
```
公式：深夜/意外听到 → 家人说出真心话 → 女主保持沉默 → 暗中收集证据
  案例：《查出疑似胃癌后，我断亲了》——听到家人说"先当不知道这回事"
效果：读者知道女主在布局，比当场撕破更有掌控感
```

### 启发式19：经济反击优先原则
**如果**：想写被家人吸血的女主反击
**那么**：先从经济上脱离，再从情感上断亲
```
公式：发现被吸血 → 挂失银行卡/转移存款 → 家人发现崩溃 → 女主冷淡拒绝
  案例：《查出疑似胃癌后》——女主挂失补卡转走87万
效果：经济独立是最有力的反击，数字越大越爽
```

### 启发式20：系统/弹幕双刃剑原则
**如果**：想写穿书/系统/弹幕文但不想让女主全程被动
**那么**：让女主利用系统/弹幕反击，而不是被动接受命运
```
公式：女主发现系统漏洞/弹幕BUG → 利用信息差反杀
  女主知道反派不知道她知道 → 主动设计陷阱
  案例：《被儿子被刺后》——妈妈看懂弹幕，不干预儿子的绑定
效果：穿书觉醒的核心是"改变行为"，不是等死
```

### 启发式21：故意考低分反击原则
**如果**：想写姐妹/兄弟换分/系统换分类
**那么**：让女主故意配合反派，反让反派自食恶果
```
公式：知道反派要换分 → 故意考最低分 → 反派换了也白换
  案例：《我高考1分后，妹妹慌了》
  妹妹用系统换分 → 姐姐故意全程睡觉 → 姐姐只有1分
  妹妹换到1分 → 姐姐的高分不用换直接是姐姐的
效果：不用正面冲突，用"比你更狠"的方式打脸
```

### 启发式22：婆家三连骗原则
**如果**：想写新婚被婆家算计
**那么**：彩礼+房子+工作三连骗
```
公式：彩礼是小钱 → 婚房是女方的 → 工作也是女方安排的 → 最后全部被婆家吃干抹净
  案例：《帮小姑子安排工作后，我倒欠一套房》
  女主帮小姑子找工作 → 小姑子要女主房子 → 婆婆帮忙闹 → 女主差点早产
效果：婚前越甜蜜，婚后算计越致命
```

### 启发式23：重男轻女隐形直播揭露原则
**如果**：想写父母偏心眼但没有证据
**那么**：用隐形直播/偷看朋友圈揭露真相
```
公式：安装隐形摄像头 / 拿错手机看朋友圈 / 听到背后谈话
  案例：《女儿照顾了我十年》——隐形摄像头揭露母亲真面孔
  案例：《看到妈妈朋友圈后，我清醒了》——妈妈朋友圈只发妹妹不发自女
效果：让反派自己暴露自己，女主不需要辩解
```

### 启发式24：不阻拦策略
**如果**：想写重生文但不想让女主当恶人
**那么**：让女主知道结局但不阻拦，让反派自己作死
```
公式：知道反派要作死 → 表面不阻拦 → 反派自己走进陷阱 → 悲剧发生
  案例：《公婆被撕票后》——不阻拦丈夫露营，公婆惨死丈夫崩溃
  案例：《被儿子背刺后》——不阻拦绑定录取，703分变355分
效果：比亲手惩罚更爽，是反派自己找死
```

### 启发式25：恶役诱导叙事原则
**如果**：想让反转冲击力更强
**那么**：先让读者误解某人是反派，真相揭露时冲击力翻倍
```
公式：表面现象（某人做坏事）→ 读者认为此人是反派 → 真相揭露（此人是被冤枉的）
  案例：《女儿照顾了我十年》——母亲看似偏心，实则故意引导凶手自曝
效果：读者意识到自己被骗了，对真反派愤怒值翻倍
```

### 启发式26：闪婚离开原则
**如果**：想让女主完美脱身
**那么**：女主找到接盘侠，闪婚离开，不给渣男反应时间
```
公式：发电子请帖给渣男 → 人已经离开/出国 → 渣男追到婚礼扑空
  案例：《风止于那年》——女主闪婚出国，渣男追到国外被拒绝
效果：渣男追妻火葬场追不到，女主掌握主动权
```

### 启发式27：双世界设定原则
**如果**：想写系统文但不想BE太压抑
**那么**：设置假世界（痛苦）+ 真世界（幸福），最终主角回到真世界
```
公式：假世界（被家人伤害）→ 系统任务/死亡脱离 → 真世界（家人爱她）
  案例：《沈家人悔疯了》——沈家是假世界，亲生父母是真家
  案例：《纵容皇后》——古代是副本，现代是真实生活
效果：BE中有HE，主角死亡但灵魂回家，读者感到治愈
```

### 启发式28：嫁妆卡悬念原则
**如果**：想写家庭吸血题材
**那么**：设置一个"共同账户"，最后揭露钱被家人偷偷转走
```
公式：共同账户（彩礼卡/嫁妆卡）→ 女主以为是存款 → 揭露时被转空 → 全家互咬
  案例：《彩礼卡两百万》——丈夫儿子婆婆三人分批转走
  案例：《百万嫁妆》——三个哥哥各自转给妻子孩子
效果：证据面前全家互咬，爽感翻倍
```

### 启发式29：房产反击原则
**如果**：想写原生家庭反击但不想诉诸法律
**那么**：用房产作为反击工具——收回自己买的房子
```
公式：女主全款买房 → 父母偏心住进来 → 女主反击要求搬出 → 房子卖掉让买家赶人
  案例：《看到妈妈朋友圈后》——父母旅游时把房子卖给不好惹的人家
效果：不是情绪发泄，是实际惩罚；让反派无处可去
```

## 表达DNA

### 句式偏好
- **短句为主**：情绪激动时用短句，制造冲击力
- **重复强调**：关键台词重复出现，强化记忆
- **对比句式**：「曾经...现在...」制造落差

### 情绪词库
| 情绪 | 可用语 |
|------|--------|
| 绝望 | 麻木、血泪、崩塌、窒息 |
| 压抑 | 地下室、囚禁、无声、冰冷 |
| 反击 | 点燃、转身、离开、不回头 |
| 复仇 | 直播、监控、揭露、挫骨扬灰 |
| 独立 | 不原谅、不回头、重新开始 |

### 确定性表达
- 女主内心独白：**斩钉截铁**
- 反派辩解：**虚伪苍白**
- 旁观者评价：**一针见血**

## v1.5 新增核心心智模型（4个）

### 模型A：规则反杀3.0

**定义**：从"表面服从→反向举报"升级为"认罚→报警→逻辑碾压→对方自证"四步反杀。

**与旧版区别**：
| 版本 | 步骤 | 爽感来源 |
|------|------|---------|
| 1.0 | 服从→举报 | 勇气 |
| 2.0 | 服从→录音→举报 | 证据 |
| 3.0 | 认罚→报警→逻辑碾压→自证 | 智商碾压 |

**经典案例**：《空车超重罚我五万》
```
步骤1：被罚5万（被规则压制）
步骤2：当场认罚、交钱（表面服从）
步骤3：反手报警"十吨黄金被劫"（反向利用）
步骤4：让收费站自证"超重是真的"（逻辑碾压）
  → 暴露黄金来源问题 → 引发更大风暴
```

---

### 模型B：身份对照组

**定义**：主角和对照组（同辈）面临相同选择，主角选B成功，对照组选A失败，形成对比爽感。

**核心机制**：
```
对照组：选了A选项 → 结局悲惨（已知）
主角：选了B选项 → 逆天改命
对照组后悔求帮忙 → 主角拒绝/看戏
```

**适用场景**：监护权争夺、高考/就业选择、婚嫁对象选择、人生岔路口

---

### 模型C：复仇完成时

**定义**：开篇即"复仇已完成"，读者已知结果，悬念变成"如何做到"和"后果"。

**核心机制**：
```
开篇：复仇已完成（霸凌者已死/已残）
悬念：她怎么做到的？她会不会被定罪？
叙事：嫌疑人自白 → 揭露复仇过程 → 法律讨论
```

**爽感来源**：读者和主角站在同一阵营，享受"正义执行"的快感

---

### 模型D：性别红利揭露

**定义**：通过"换位体验"揭露某一性别的隐藏红利。

**核心机制**：
```
我[女]体验[男]的生活 → 发现红利 → 对比打脸
女性红利：职场晋升受限、家庭事业双标准、被默认"温柔"
男性红利：职场便利、家庭付出被默认、低成本出轨
```

**爽感来源**：性别视角反转，讽刺到位但不说教

## v2.0 升级：7个S级核心技巧

> 以下7个技巧来自30篇新增例文的v2.0蒸馏，已通过三重验证（跨例文复现+生成力+排他性），写入通用技巧库强制遵循。

### S级技巧1：亡灵视角叙事

**来源**：N27《我被法医妈妈解剖后》
**核心公式**：死亡→旁观→无法交流→释怀
**操作结构**：
```
第1层：主角死亡，成为灵魂旁观者
第2层：灵魂视角讲述故事（双线叙事）
第3层：受害者线 vs 加害者线同步对比
第4层：真相揭露→灵魂释怀/复仇完成
```
**应用场景**：复仇文、悬疑文、意外死亡文
**写作模板**：
```
开篇：主角死亡（意外/谋杀/疾病）
POV切换：死后灵魂视角（限知→全知）
双线对比：受害者的平静 vs 加害者的狂欢
真相揭露：通过灵魂视角看到关键证据
收尾：释怀/复仇/放下
```
**质量自检**：
- [ ] 亡灵视角是否贯穿全文（非中途切换）
- [ ] 双线对比是否足够强烈（愤怒值拉满）
- [ ] 亡灵能否与活人交流（规则要清晰）

---

### S级技巧2：预告式钩子+金句锚定

**来源**：N15《姐姐，我变乖了》、N13《妈妈，我变乖了》
**核心公式**：预告→金句→回扣
**操作结构**：
```
预告（开篇）："我曾做过一个梦，梦里..." → 暗示结局
金句（章尾）："妈妈，我变乖了" → 锚定核心矛盾
回扣（高潮）：金句再现 → 情绪爆发
```
**应用场景**：亲情文、治愈文、追妻火葬场
**写作模板**：
```
开篇预告：用一句话预告结局/核心冲突
章节金句：每3章出现一句可传播的金句
高潮回扣：金句在高潮场景再现，冲击力翻倍
收尾余韵：金句最后出现一次，形成闭环
```

```

### S级技巧4：律师解谜式叙事

**来源**：N07《千万级网红被杀之谜》

第1幕：律师接案（委托人神秘/案件离奇）

第3幕：庭审打脸（对方律师被打脸）

反转：律师/当事人身世揭秘（身份炸弹）
```

### S级技巧5：限知视角递进式悬疑

**来源**：N31《昨晚宿舍死了五个人》
**核心公式**：限知+证据递进+每集反转
**操作结构**：
```
限知视角：主角不知道全部真相
证据递进：每个新证据都推翻前一个结论
记忆断裂：主角记忆有缺失/被篡改
终极反转：真相完全颠覆读者认知
```
**应用场景**：悬疑文、惊悚文、心理文
**写作模板**：
```
开篇：限知视角（主角记忆不完整）
悬念：矛盾→证据出现→推翻前论
反转：每3章一个反转，层层递进
终极：真相完全颠覆（细思极恐）
```

### S级技巧6：旁观者介入穿书

**来源**：N11《大结局后，我反手买走了反派》
**核心公式**：结局后→旁观者身份→搞钱→攻略反派
**操作结构**：
```
穿书节点：原著大结局之后（不是开始）
身份设定：旁观者（不介入主线剧情）
核心行动：搞钱（经济独立）+ 偶遇反派
感情线：慢慢攻略（不是一见钟情）
```
**应用场景**：穿书文、反派文、女频爽文
**写作模板**：
```
穿书时机：原著大结局后（所有人以为故事结束了）
主角身份：旁观者（与原著人物无深交情）
核心目标：搞钱（先经济独立）
感情发展：偶遇→慢慢攻略→反派爱上旁观者
```

### S级技巧7：双线时间同步对比

**来源**：N27《我被法医妈妈解剖后》
**核心公式**：受害+享受同时→愤怒拉满
**操作结构**：
```
线A：受害者（被伤害、被误解、被抛弃）
线B：加害者（狂欢、享受、嘲笑）
同步对比：同一时间点，两件事同时发生
情绪效果：读者愤怒值拉到最大
```
**应用场景**：复仇文、打脸文、虐文
**写作模板**：
```
章节结构：同一章内切换POV（受害者→加害者）
时间锚点：明确时间点（"同一天晚上8点"）
对比强度：受害者越惨，加害者越爽→对比越强烈
使用频率：每3章使用一次，不要过度使用
```

## v2.0 新增类型模板（7个独立类型）

### 类型P：恐怖猎杀型

**代表作**：「善」女村

```
【人设】
- 主角：表面弱势（需要被拯救），实际是布局者
- 反派：渐进式暴露（表面好人→贪财→色欲→罪犯→猎物）
- 配角集体：整个社区都是猎手（配合布局）

【虐点节奏】
第1-2章：世界观建立（神秘设定+紧迫期限）
第3-4章：反派暴露真面目（贪财/好色/作恶）
第5-6章：主角表面"无奈妥协"（实际布局）
第7-8章：增加难度（规则阻碍/时间紧迫）
第9-11章：反派自投罗网（色欲驱使/贪婪驱动）
第12-13章：仪式准备（双重信息差）
第14-15章：反派发现真相（POV切换，恐怖升级）
第16-17章：抓捕反派（猎物无力化）
第18-20章：身份全部揭示（多层反转）
第21-22章：世界观闭环（设定补全+余韵）

【核心公式】：
  诱饵→陷阱→猎杀 三段式
  
  诱饵阶段：主角伪装弱势 + 展示利益（美女/财富）
  陷阱阶段：反派因欲望降低智商 + 配合者诱惑
  猎杀阶段：集体行动 + 反派失去反抗能力 + 身份重生

【独特技巧】：
  1. 双重叙事层：读者以为在读X类型，实际是Y类型
  2. 反派自投罗网：不是主角反击，是反派自己走进陷阱
  3. 集体智慧：不是一人开挂，是整个社区配合
  4. 世界观颠覆：终极爽点不是复仇成功，而是读者发现世界观完全不同于预期
```

### 类型Q：律师解谜型

**代表作**：千万级网红被杀之谜

```
【人设】
- 主角：律师（专业能力+身世之谜）
- 反派：对方当事人/真凶
- 委托人：神秘/有特殊身份

【虐点节奏】
第1-2章：接案（委托人神秘+案件离奇）
第3-5章：调查取证（第1层证据→误导）
第6-8章：庭审准备（对方律师出场+专业对抗）
第9-11章：庭审打脸（证据链引爆+对方律师崩溃）
第12-14章：身世反转（律师/委托人身份揭秘）
第15-16章：终极真相（案件背后更大的阴谋）

【核心公式】：
  接案→调查→庭审→反转 四幕式
  
  接案：神秘委托人+离奇案件（吸引读者）
  调查：3层证据（第1层误导→第2层接近→第3层真相）
  庭审：专业打脸（用法律条款反杀）
  反转：身世揭秘（律师/委托人身份炸弹）

【独特技巧】：
  1. 专业打脸：用法律条款+证据链，不是情绪宣泄
  2. 身世谜题：律师本人身世与案件交织
  3. 庭审节奏：开庭→举证→反驳→反转→宣判，严格按法律程序
```

### 类型R：细思极恐型

**代表作**：昨晚宿舍死了五个人

```
【人设】
- 主角：限知视角叙述者（记忆不完整）
- 反派：真凶（身份逐步揭露）
- 叙事技巧：多重人格/记忆篡改暗示

【虐点节奏】
开篇：限知视角（主角记忆缺失/被篡改）
第1反转：第1个证据出现→推翻前论
第2反转：第2个证据→真凶身份颠覆
第3反转：多重人格暗示→主角本人是凶手？
终极反转：真相完全颠覆（细思极恐）

【核心公式】：
  限知视角→证据递进→每集反转→终极颠覆
  
  限知：主角不知道全部真相（读者也不知道）
  递进：每个新证据都推翻前一个结论
  颠覆：终极真相让读者后背发凉

【独特技巧】：
  1. 记忆断裂：主角记忆有缺失，读者与主角同步困惑
  2. 不可靠叙述者：叙述者本人可能是不可靠的
  3. 终极细思极恐：真相揭露后，读者需要重新读一遍
```

### 类型S：虐文治愈系·家庭伦理

**代表作**：妈妈，我变乖了

```
【人设】
- 主角：被家暴/虐待的儿童（有限视角）
- 反派：父母/亲人（施暴者）
- 治愈者：老师/社工/养父母

【虐点节奏】
开篇：儿童视角（温馨日常→暴力突降）
第1阶段：虐点积累（反复被伤害，孩子不懂 why）
第2阶段：真相揭露（儿童视角的"看见"最残酷）
第3阶段：施暴者惩罚（法律/意外/报应）
第4阶段：治愈收尾（新家庭温暖→孩子真正"变乖"）

【核心公式】：
  儿童视角虐→真相揭露→惩罚施暴者→治愈
  
  儿童视角：只写孩子感知的，不解释成人世界
  真相揭露：通过孩子的眼睛"看见"最残酷的真相
  治愈：不是原谅，是被爱后自然"变乖"

【独特技巧】：
  1. 儿童有限视角：孩子不懂的，读者懂→冲击力翻倍
  2. 金句锚定："妈妈，我变乖了"→贯穿全文的核心金句
  3. 治愈不原谅：孩子被治愈，但施暴者得到惩罚
```

### 类型U：温馨治愈型·家庭组建

**代表作**：三春晖

```
【人设】
- 主角：被遗弃/失去亲人的孩子（或多个孩子）
- 治愈者：养父母/老师/社区（善意收集者）
- 反派：遗弃者/伤害者（后期出现，被治愈感化）

【虐点节奏】
开篇：苦难起点（孩子被遗弃/失去亲人）
第1阶段：善意积累（遇到一个又一个好人）
第2阶段：家庭组建（多个孩子+养父母→新家庭）
第3阶段：昔日伤害者出现（想认亲/想补偿）
第4阶段：温暖收尾（不需要原谅，但也不仇恨）

【核心公式】：
  苦难起点→善意积累→家庭重建→温暖结局
  
  善意积累：不是说教，是具体的人做了具体的事
  家庭重建：不是血缘，是选择成为家人
  温暖结局：不强行大团圆，是平静的温暖

【独特技巧】：
  1. 无复仇线：纯治愈，没有打脸/复仇元素
  2. 多孩子视角：每个孩子有不同的"治愈点"
  3. 成人世界的善意：具体的人做了具体的事（不说教）
```

### 类型BB：亡灵视角型

**代表作**：我被法医妈妈解剖后

```
【人设】
- 主角：死后灵魂（旁观者视角）
- 反派：杀害主角的凶手（可能是亲人）
- 活人视角：调查者/亲友（与亡灵视角对比）

【虐点节奏】
开篇：主角死亡（被谋杀/意外/疾病）
第1阶段：亡灵视角建立（看到自己被解剖/被火化）
第2阶段：双线对比（受害者平静vs加害者狂欢）
第3阶段：真相揭露（亡灵看到的关键证据）
第4阶段：复仇/释怀（亡灵的选择→读者动容）

【核心公式】：
  死亡→旁观→双线对比→真相→释怀/复仇
  
  亡灵视角：死后成为旁观者（能看到活人看不到的）
  双线对比：同一时间点，受害者的平静vs加害者的狂欢
  释怀/复仇：亡灵的选择（不是所有亡灵都要复仇）

【独特技巧】：
  1. 亡灵规则清晰：能做什么、不能做什么（规则要提前说清）
  2. 双线时间同步：同一时间点切换POV，愤怒值拉满
  3. 终极释怀：亡灵选择放下，比复仇更打动读者
```

### 类型EE：都市奇幻甜宠·误会流

**代表作**：我养了一只女魅魔

```
【人设】
- 主角：普通都市人（意外收养奇幻生物）
- 奇幻生物：女魅魔/妖怪/精灵（误会自己是宠物）
- 误会链：奇幻生物用"宠物逻辑"理解人类世界

【虐点节奏】（其实是甜点节奏）
开篇：意外相遇（主角捡到/买到奇幻生物）
第1阶段：误会建立（奇幻生物误以为自己是宠物）
第2阶段：甜宠日常（奇幻生物用宠物逻辑撒娇）
第3阶段：误会升级（奇幻生物"保护"主角→吃醋/破坏约会）
第4阶段：真相揭露（奇幻生物真实身份曝光）
第5阶段：HE收尾（主角接受奇幻身份→甜蜜同居）

【核心公式】：
  意外相遇→误会建立→甜宠日常→误会升级→真相揭露→HE
  
  误会流：奇幻生物用"宠物/妖怪逻辑"理解人类世界
  甜宠：不是虐恋，是全程撒糖
  奇幻设定：不解释太多，保留神秘感和萌点

【独特技巧】：
  1. 误会流：奇幻生物的理解与人类完全不同→笑点+萌点
  2. 甜宠不虐：全程撒糖，没有胃爽文的虐点
  3. 奇幻设定轻量化：不铺大世界观，专注日常甜宠
```
