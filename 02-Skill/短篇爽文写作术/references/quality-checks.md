# 质量校验清单

> 短篇爽文写作术 · 子文件
> 主文件 SKILL.md 中引用，校验文章质量时加载。

---

## 通用写作检查清单

### 开篇检查（必须满足）
- [ ] 第一句有炸弹
- [ ] 女主困境清晰
- [ ] 反派身份明确
- [ ] 读者情绪被抓住

### 中段检查（每3章检查）
- [ ] 有新的加害/虐点
- [ ] 有微小的希望（贵人/证据）
- [ ] 反派有内讧迹象
- [ ] 女主有内心成长

### 高潮检查（第6-7章）
- [ ] 证据一次性引爆
- [ ] 反派当众社死
- [ ] 女主掌握主动权
- [ ] 读者情绪释放

### 结尾检查
- [ ] 女主不依附反派
- [ ] 反派得到应有惩罚
- [ ] 女主有新生活/新感情
- [ ] 读者感到满足

---

## 规则反杀文校验
- [ ] 主角被规则压制的理由充分
- [ ] 主角"认罚"让对方放松警惕
- [ ] 反向操作有法律/逻辑依据
- [ ] 对方"自证"过程有说服力
- [ ] 数字量化（金额/重量）增加可信度

## 身份逆袭文校验
- [ ] 身份对比足够强烈（假→真）
- [ ] 对照组的错误选择有逻辑
- [ ] 对照组见证主角成功的场景够震撼
- [ ] 主角选择B的决策有智慧
- [ ] 不需要强行原谅任何人

## 性别反转文校验
- [ ] 性别红利揭露自然，不说教
- [ ] 主角本身是强者（有钱/有权）
- [ ] 甜蜜日常足够撒糖
- [ ] 无需复仇，全程轻松
- [ ] 女性视角讽刺到位

## 悬疑复仇文校验
- [ ] 复仇完成时开篇足够震撼
- [ ] 嫌疑人自白角度新颖
- [ ] 法律vs复仇的讨论有深度
- [ ] 主角不后悔的态度坚定
- [ ] 读者和主角站在同一阵营
