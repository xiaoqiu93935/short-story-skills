## 基础启发式（1-29）

### 启发式1：开篇炸弹原则
**如果**：你不知道从哪里开始写
**那么**：第一句话就要扔出炸弹
```
公式：地点+身份+炸弹内容
示例："从精神病院出来当天，丈夫傅云峥毫无征兆地开口..."
```

### 启发式2：三连背叛原则
**如果**：想写最极致的虐
**那么**：让丈夫+父亲+闺蜜同时背叛
```
案例：《繁华未尽》傅云峥(丈夫)+梁父(父亲)+梁思瑶(替代者)
```

### 启发式3：证据链原则
**如果**：想让打脸有说服力
**那么**：提前埋设证据，最后一次性引爆
```
证据类型：监控录像/录音/聊天记录/转账记录/精神科鉴定
```

### 启发式4：直播审判原则
**如果**：想让反派社死
**那么**：让反派在公众场合自己作死
```
场景：直播审判/采访/发布会/婚礼
```

### 启发式5：备胎守护原则
**如果**：女主太惨需要希望
**那么**：设置一个默默守护的青梅竹马/哥哥/陌生人
```
案例：《深夜电台》谢年
功能：接女主脱离苦海，但不是立刻出现
```

### 启发式6：反派内讧原则
**如果**：想让反派倒台
**那么**：让反派集团内部先打起来
```
案例：《繁华未尽》傅云峥报警抓梁思瑶
```

### 启发式7：疾病/心理疾病反转原则
**如果**：想写三角关系
**那么**：让"有心理疾病"的人成为加害者
```
案例：《风止于那年》唐紫宁假抑郁控制沈怀川
```

### 启发式8：学历/学位剥夺原则
**如果**：想写职场/学术界爽点
**那么**：让反派被撤销学历/学位/执业资格
```
案例：《深夜电台》夏月学位证被撤销
```

### 启发式9：原谅羞耻原则
**如果**：读者想看女主强硬
**那么**：女主选择不原谅，且理由充分
```
公式：我不是不原谅，是不值得原谅
```

### 启发式10：标题意象原则
**如果**：不知道标题怎么取
**那么**：用「X未尽，Y无计」公式
```
公式1：[情感词]+未尽，[主题词]+无计
  繁华未尽，此情无计
公式2：[身份]+[困境]
  笼中雀，局中人
公式3：[动词]+[意象]
  花开不候，折柳无情
```

### 启发式11：对仗反击标题原则
**如果**：想让标题本身有爽感
**那么**：用「你X，我Y」对仗结构
```
公式：你[做某事]，我[反手反击]
  你娶平妻，我纳面首不过分吧
  你种我瓜，我收你田
效果：标题即预告，读者一看就知道有反击
```

### 启发式12：疑问感叹标题原则
**如果**：想用现代题材
**那么**：用「[事件]？[态度]，[反手行动]：[结果]」结构
```
公式：[被欺压的事]？[态度]，[反手行动]：[反转结果]
  空车超重罚我五万？我认罚，反手报警：十吨黄金被劫
  领导抢我方案？我截图，转手发工作群
效果：标题完整叙事，读者一看就知道爽点在哪
```

### 启发式13：羡慕嫉妒开场原则
**如果**：想写性别/职场讽刺文
**那么**：用「我嫉妒他/她」开场建立对比
```
公式：我嫉妒[拥有某特权的某群体]
  → 揭示他们享有的隐藏红利
  → 反转/换位后打脸
  我嫉妒我三十多岁的男同事（因为老婆全职带娃）
  我嫉妒我女上司（因为老公有钱）
效果：自然引入核心矛盾，读者容易代入
```

### 启发式14：悬疑复仇完成时原则
**如果**：想写复仇文但不想写具体作案过程
**那么**：开篇即「复仇已完成」，主角被当成嫌疑人
```
公式：[悲剧发生]后，我[平静/微笑/毫不在意]
  霸凌我的人接二连三的死于非命后，我被列为了头号嫌疑人。
  我笑了。我等这一天好久了。
效果：开篇即高潮，读者知道复仇已完成
  → 悬念变成「她怎么做到的」「她会不会被定罪」
```

### 启发式15：时间差信息差原则
**如果**：想写亲人背叛但反派不知情
**那么**：利用「时间差」制造信息差
```
公式：女主已知真相N年 → 反派以为一切如常 → 女主平静揭露
  女主：爸已经去世7年了。
  反派：（震惊）什么？不可能！
  女主：我亲眼看着碑立起来的。
效果：不需要激烈冲突，用平静陈述制造最强冲击力
```

### 启发式16：参赛圆梦不原谅原则
**如果**：想让女主完美收尾但保持人设
**那么**：让女主自我实现（参赛/事业）+ 不原谅反派
```
公式：参赛成功（事业线） + 拒绝和解（人设线）
  女主赢得银奖 → 反派求见面 → 女主：我不需要你的原谅
效果：事业线和复仇线同时完成，不相互稀释
```

### 启发式17：亲情背叛叠加利益背叛原则
**如果**：想写极致背叛感
**那么**：亲情背叛（害死父亲）+ 利益背叛（夺走前途）+ 信息背叛（隐瞒死讯）三合一
```
三层背叛：
第1层：亲人为爱情害死父亲
第2层：亲人用女主前途换取户口本
第3层：亲人7年不知父亲死讯（被配偶隐瞒）
爽感：三层叠加让原谅变得不可能
```

### 启发式18：偷听+假装不知道原则
**如果**：想让女主发现家人真面目
**那么**：让女主偶然听到家人背后对话，但先不揭穿
```
公式：深夜/意外听到 → 家人说出真心话 → 女主保持沉默 → 暗中收集证据
  案例：《查出疑似胃癌后，我断亲了》——听到家人说"先当不知道这回事"
效果：读者知道女主在布局，比当场撕破更有掌控感
```

### 启发式19：经济反击优先原则
**如果**：想写被家人吸血的女主反击
**那么**：先从经济上脱离，再从情感上断亲
```
公式：发现被吸血 → 挂失银行卡/转移存款 → 家人发现崩溃 → 女主冷淡拒绝
  案例：《查出疑似胃癌后》——女主挂失补卡转走87万
效果：经济独立是最有力的反击，数字越大越爽
```

### 启发式20：系统/弹幕双刃剑原则
**如果**：想写穿书/系统/弹幕文但不想让女主全程被动
**那么**：让女主利用系统/弹幕反击，而不是被动接受命运
```
公式：女主发现系统漏洞/弹幕BUG → 利用信息差反杀
  女主知道反派不知道她知道 → 主动设计陷阱
  案例：《被儿子被刺后》——妈妈看懂弹幕，不干预儿子的绑定
效果：穿书觉醒的核心是"改变行为"，不是等死
```

### 启发式21：故意考低分反击原则
**如果**：想写姐妹/兄弟换分/系统换分类
**那么**：让女主故意配合反派，反让反派自食恶果
```
公式：知道反派要换分 → 故意考最低分 → 反派换了也白换
  案例：《我高考1分后，妹妹慌了》
  妹妹用系统换分 → 姐姐故意全程睡觉 → 姐姐只有1分
  妹妹换到1分 → 姐姐的高分不用换直接是姐姐的
效果：不用正面冲突，用"比你更狠"的方式打脸
```

### 启发式22：婆家三连骗原则
**如果**：想写新婚被婆家算计
**那么**：彩礼+房子+工作三连骗
```
公式：彩礼是小钱 → 婚房是女方的 → 工作也是女方安排的 → 最后全部被婆家吃干抹净
  案例：《帮小姑子安排工作后，我倒欠一套房》
  女主帮小姑子找工作 → 小姑子要女主房子 → 婆婆帮忙闹 → 女主差点早产
效果：婚前越甜蜜，婚后算计越致命
```

### 启发式23：重男轻女隐形直播揭露原则
**如果**：想写父母偏心眼但没有证据
**那么**：用隐形直播/偷看朋友圈揭露真相
```
公式：安装隐形摄像头 / 拿错手机看朋友圈 / 听到背后谈话
  案例：《女儿照顾了我十年》——隐形摄像头揭露母亲真面孔
  案例：《看到妈妈朋友圈后，我清醒了》——妈妈朋友圈只发妹妹不发自女
效果：让反派自己暴露自己，女主不需要辩解
```

### 启发式24：不阻拦策略
**如果**：想写重生文但不想让女主当恶人
**那么**：让女主知道结局但不阻拦，让反派自己作死
```
公式：知道反派要作死 → 表面不阻拦 → 反派自己走进陷阱 → 悲剧发生
  案例：《公婆被撕票后》——不阻拦丈夫露营，公婆惨死丈夫崩溃
  案例：《被儿子背刺后》——不阻拦绑定录取，703分变355分
效果：比亲手惩罚更爽，是反派自己找死
```

### 启发式25：恶役诱导叙事原则
**如果**：想让反转冲击力更强
**那么**：先让读者误解某人是反派，真相揭露时冲击力翻倍
```
公式：表面现象（某人做坏事）→ 读者认为此人是反派 → 真相揭露（此人是被冤枉的）
  案例：《女儿照顾了我十年》——母亲看似偏心，实则故意引导凶手自曝
效果：读者意识到自己被骗了，对真反派愤怒值翻倍
```

### 启发式26：闪婚离开原则
**如果**：想让女主完美脱身
**那么**：女主找到接盘侠，闪婚离开，不给渣男反应时间
```
公式：发电子请帖给渣男 → 人已经离开/出国 → 渣男追到婚礼扑空
  案例：《风止于那年》——女主闪婚出国，渣男追到国外被拒绝
效果：渣男追妻火葬场追不到，女主掌握主动权
```

### 启发式27：双世界设定原则
**如果**：想写系统文但不想BE太压抑
**那么**：设置假世界（痛苦）+ 真世界（幸福），最终主角回到真世界
```
公式：假世界（被家人伤害）→ 系统任务/死亡脱离 → 真世界（家人爱她）
  案例：《沈家人悔疯了》——沈家是假世界，亲生父母是真家
  案例：《纵容皇后》——古代是副本，现代是真实生活
效果：BE中有HE，主角死亡但灵魂回家，读者感到治愈
```

### 启发式28：嫁妆卡悬念原则
**如果**：想写家庭吸血题材
**那么**：设置一个"共同账户"，最后揭露钱被家人偷偷转走
```
公式：共同账户（彩礼卡/嫁妆卡）→ 女主以为是存款 → 揭露时被转空 → 全家互咬
  案例：《彩礼卡两百万》——丈夫儿子婆婆三人分批转走
  案例：《百万嫁妆》——三个哥哥各自转给妻子孩子
效果：证据面前全家互咬，爽感翻倍
```

### 启发式29：房产反击原则
**如果**：想写原生家庭反击但不想诉诸法律
**那么**：用房产作为反击工具——收回自己买的房子
```
公式：女主全款买房 → 父母偏心住进来 → 女主反击要求搬出 → 房子卖掉让买家赶人
  案例：《看到妈妈朋友圈后》——父母旅游时把房子卖给不好惹的人家
效果：不是情绪发泄，是实际惩罚；让反派无处可去
```

## v1.5 新增启发式（30-40）

### 启发式30：京圈身份碾压原则
**如果**：想写豪门/权贵题材
**那么**：让主角的真实身份（公主/首富女儿/权贵继女）比所有人想象的都高
```
公式：所有人以为主角是[低身份] → 揭露真相是[高身份] → 权力碾压
  案例：假继女 → 真京圈皇太女继女
```

### 启发式31：政治线打脸原则
**如果**：想写古言/权谋题材
**那么**：加入政治势力博弈，主角的靠山比反派更大
```
公式：反派以为靠山是[左相] → 主角靠山是[皇帝/公主] → 政治碾压
```

### 启发式32：复仇完成时叙事原则
**如果**：想写悬疑复仇但不想铺悬念
**那么**：开篇即"复仇已完成"，读者已知结果，悬念变成"如何做到"
```
公式：[悲剧发生]后，我被列为嫌疑人/我笑了
```

### 启发式33：嫌疑人自白原则
**如果**：想让复仇叙事更震撼
**那么**：让主角以"嫌疑人/被捕者"身份自白，揭露复仇过程
```
公式：被审讯 → 主角平静讲述复仇过程 → 审讯者震惊
```

### 启发式34：法律与复仇讨论原则
**如果**：想让复仇文有深度
**那么**：加入"法律vs复仇"的道德讨论
```
公式：主角复仇 → 被问"你不相信法律吗" → 主角揭露法律无法制裁的原因
```

### 启发式35：长篇现实爽文节奏原则
**如果**：想写万字以上现实题材爽文
**那么**：用"小事件→大事件→终极反转"的多层结构
```
公式：事件1（500字）→ 事件2（2000字）→ 事件3（5000字）→ 高潮
```

### 启发式36：认罚即反杀原则
**如果**：想让规则反杀更高级
**那么**：主角主动"认罚"让对方放松警惕，再突然反击
```
公式：被罚 → 主角："我认罚，交钱" → 反派得意 → 主角："但我要报警"
```

### 启发式37：性别红利揭露原则
**如果**：想写性别讽刺/女尊题材
**那么**：通过"换位体验"揭露某一性别的隐藏红利
```
公式：我[女]体验[男]的生活 → 发现红利 → 对比打脸
```

### 启发式38：甜蜜日常无虐原则
**如果**：想写甜文不想写虐
**那么**：主角本身已经是强者（有钱/有权），不需要复仇
```
公式：主角是[金主/强者] → 找[老婆/老公] → 甜蜜日常
```

### 启发式39：对照组见证后悔原则
**如果**：想让身份逆袭更爽
**那么**：让选了错误路线的对照组亲眼见证主角的成功
```
公式：对照组选了A → 结局悲惨
  主角选了B → 成功逆袭
  对照组出现求帮忙 → 主角拒绝/看戏
```

### 启发式40：继母/继女关系逆袭原则
**如果**：想写现代豪门家庭剧
**那么**：用"继女"身份作为切入点，继母可能是盟友也可能是敌人
```
公式：假继女身份 → 真豪门血脉 → 继母态度转变
```
