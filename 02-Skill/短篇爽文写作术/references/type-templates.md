## 类型A：隐忍反击型
**代表作**：《繁华未尽，此情无计》
**核心矛盾**：至亲联手背叛→女主隐忍收集证据→消失→反派内讧→真相大白

```
【人设】
- 女主：被至亲背叛的隐忍者
- 反派：私生女/替代者 + 帮凶至亲
- 观众：不知情的第三方

【虐点节奏】
第1章：炸弹揭露（背叛真相）
第2章：多次加害（精神折磨）
第3章：最低谷（被迫签字/澄清）
第4章：假死/消失
第5章：反派视角崩溃
第6章：证据曝光
第7章：反派受惩罚
第8章：独立收尾
```

第3步：反向操作（报警/举报/曝光）
第4步：逻辑碾压（对方用自己证据自证）
收尾：小标题数字分段（01 空气十吨重）

```

【节奏波形】
开篇：监护权/人生岔路选择（已知结局）
对比：上辈子选X的结局 + 这辈子选Y的走向
爽点：对照组选错后悔 + 主角选对逆袭
收尾：对照组求主角帮忙 / 主角彻底独立
```

## 类型I：亲情断绝型
**代表作**：《与岁同安，与时决晏》
**核心矛盾**：亲人背叛→断绝关系→多年独立→真相揭露→不原谅收尾

```
【人设】
- 女主：被亲人（兄/姐）背叛后的独立者
- 反派：亲人 + 帮凶（外人/配偶）
- 武器：时间（自我重建）+ 真相揭露

【虐点节奏】
开篇：春节/节日团聚场景 + 前任突然出现
第1段：女主平静生活（丈夫+孩子）
第2段：回忆背叛（亲情如何被背叛）
第3段：七年断绝（父亲去世+女主重建）
第4段：哥哥归来（不知父亲已死）
第5段：真相揭露（女主平静陈述）
第6段：哥哥崩溃（离婚/求原谅）
第7段：女主参赛/自我实现（凭实力翻身）
第8章：不原谅收尾（保持距离，各自安好）

【特别收尾类型】：不强求和解，不原谅也不仇恨
  核心态度：恨太累，我只想好好过自己的日子
  爽感来源：女主不再需要反派认可，已自我圆满
```

## 类型J：世情打脸型
**代表作**：《查出疑似胃癌后，我断亲了》《我哥拿走我妈住院报销的钱，我断亲了》
**核心矛盾**：家人长期吸血→真相揭露→经济反击→彻底断亲

```
【人设】
- 女主：长期被原生家庭吸血工具人
- 反派：父母/兄弟组成的吸血鬼家族
- 武器：经济独立 + 证据收集

【虐点节奏】
开篇：日常场景（照顾家人/汇款）
第1步：发现猫腻（偷听/体检报告/朋友圈）
第2步：确认背叛（家里有钱却哭穷/弟弟吸血）
第3步：女主觉醒（停止付出）
第4步：经济反击（转移存款/挂失银行卡）
第5步：家人发现后崩溃求饶
第6步：女主拒绝回头，断亲收尾

【核心机制】：
- 反派长期PUA（"家里没钱""你是姐姐要让着"）
- 女主长期自我说服（心疼父母/为弟弟付出）
- 真相揭露：钱一直有，只是永远不会给女主
爽感：女主算清十年账，拒绝继续当工具人
```

## 类型P：恐怖猎杀型
**代表作**：「善」女村

```
【人设】
- 主角：表面弱势（需要被拯救），实际是布局者
- 反派：渐进式暴露（表面好人→贪财→色欲→罪犯→猎物）
- 配角集体：整个社区都是猎手（配合布局）

【虐点节奏】
第1-2章：世界观建立（神秘设定+紧迫期限）
第3-4章：反派暴露真面目（贪财/好色/作恶）
第5-6章：主角表面"无奈妥协"（实际布局）
第7-8章：增加难度（规则阻碍/时间紧迫）
第9-11章：反派自投罗网（色欲驱使/贪婪驱动）
第12-13章：仪式准备（双重信息差）
第14-15章：反派发现真相（POV切换，恐怖升级）
第16-17章：抓捕反派（猎物无力化）
第18-20章：身份全部揭示（多层反转）
第21-22章：世界观闭环（设定补全+余韵）

【核心公式】：
  诱饵→陷阱→猎杀 三段式
  
  诱饵阶段：主角伪装弱势 + 展示利益（美女/财富）
  陷阱阶段：反派因欲望降低智商 + 配合者诱惑
  猎杀阶段：集体行动 + 反派失去反抗能力 + 身份重生

【独特技巧】：
  1. 双重叙事层：读者以为在读X类型，实际是Y类型
  2. 反派自投罗网：不是主角反击，是反派自己走进陷阱
  3. 集体智慧：不是一人开挂，是整个社区配合
  4. 世界观颠覆：终极爽点不是复仇成功，而是读者发现世界观完全不同于预期
```

## 类型Q：律师解谜型
**代表作**：千万级网红被杀之谜

```
【人设】
- 主角：律师（专业能力+身世之谜）
- 反派：对方当事人/真凶
- 委托人：神秘/有特殊身份

【虐点节奏】
第1-2章：接案（委托人神秘+案件离奇）
第3-5章：调查取证（第1层证据→误导）
第6-8章：庭审准备（对方律师出场+专业对抗）
第9-11章：庭审打脸（证据链引爆+对方律师崩溃）
第12-14章：身世反转（律师/委托人身份揭秘）
第15-16章：终极真相（案件背后更大的阴谋）

【核心公式】：
  接案→调查→庭审→反转 四幕式
  
  接案：神秘委托人+离奇案件（吸引读者）
  调查：3层证据（第1层误导→第2层接近→第3层真相）
  庭审：专业打脸（用法律条款反杀）
  反转：身世揭秘（律师/委托人身份炸弹）

【独特技巧】：
  1. 专业打脸：用法律条款+证据链，不是情绪宣泄
  2. 身世谜题：律师本人身世与案件交织
  3. 庭审节奏：开庭→举证→反驳→反转→宣判，严格按法律程序
```

## 类型R：细思极恐型
**代表作**：昨晚宿舍死了五个人

```
【人设】
- 主角：限知视角叙述者（记忆不完整）
- 反派：真凶（身份逐步揭露）
- 叙事技巧：多重人格/记忆篡改暗示

【虐点节奏】
开篇：限知视角（主角记忆缺失/被篡改）
第1反转：第1个证据出现→推翻前论
第2反转：第2个证据→真凶身份颠覆
第3反转：多重人格暗示→主角本人是凶手？
终极反转：真相完全颠覆（细思极恐）

【核心公式】：
  限知视角→证据递进→每集反转→终极颠覆
  
  限知：主角不知道全部真相（读者也不知道）
  递进：每个新证据都推翻前一个结论
  颠覆：终极真相让读者后背发凉

【独特技巧】：
  1. 记忆断裂：主角记忆有缺失，读者与主角同步困惑
  2. 不可靠叙述者：叙述者本人可能是不可靠的
  3. 终极细思极恐：真相揭露后，读者需要重新读一遍
```

## 类型S：虐文治愈系·家庭伦理

```

【虐点节奏】
开篇：儿童视角（温馨日常→暴力突降）
第1阶段：虐点积累（反复被伤害，孩子不懂 why）
第2阶段：真相揭露（儿童视角的"看见"最残酷）
第3阶段：施暴者惩罚（法律/意外/报应）
第4阶段：治愈收尾（新家庭温暖→孩子真正"变乖"）

【核心公式】：
  儿童视角虐→真相揭露→惩罚施暴者→治愈
  
  儿童视角：只写孩子感知的，不解释成人世界
  真相揭露：通过孩子的眼睛"看见"最残酷的真相
  治愈：不是原谅，是被爱后自然"变乖"

【独特技巧】：
  1. 儿童有限视角：孩子不懂的，读者懂→冲击力翻倍
  2. 金句锚定："妈妈，我变乖了"→贯穿全文的核心金句
  3. 治愈不原谅：孩子被治愈，但施暴者得到惩罚
```

## 类型U：温馨治愈型·家庭组建
**代表作**：三春晖

```
【人设】
- 主角：被遗弃/失去亲人的孩子（或多个孩子）
- 治愈者：养父母/老师/社区（善意收集者）
- 反派：遗弃者/伤害者（后期出现，被治愈感化）

【虐点节奏】
开篇：苦难起点（孩子被遗弃/失去亲人）
第1阶段：善意积累（遇到一个又一个好人）
第2阶段：家庭组建（多个孩子+养父母→新家庭）
第3阶段：昔日伤害者出现（想认亲/想补偿）
第4阶段：温暖收尾（不需要原谅，但也不仇恨）

【核心公式】：
  苦难起点→善意积累→家庭重建→温暖结局
  
  善意积累：不是说教，是具体的人做了具体的事
  家庭重建：不是血缘，是选择成为家人
  温暖结局：不强行大团圆，是平静的温暖

【独特技巧】：
  1. 无复仇线：纯治愈，没有打脸/复仇元素
  2. 多孩子视角：每个孩子有不同的"治愈点"
  3. 成人世界的善意：具体的人做了具体的事（不说教）
```

## 类型BB：亡灵视角型
**代表作**：我被法医妈妈解剖后

```
【人设】
- 主角：死后灵魂（旁观者视角）
- 反派：杀害主角的凶手（可能是亲人）
- 活人视角：调查者/亲友（与亡灵视角对比）

【虐点节奏】
开篇：主角死亡（被谋杀/意外/疾病）
第1阶段：亡灵视角建立（看到自己被解剖/被火化）
第2阶段：双线对比（受害者平静vs加害者狂欢）
第3阶段：真相揭露（亡灵看到的关键证据）
第4阶段：复仇/释怀（亡灵的选择→读者动容）

【核心公式】：
  死亡→旁观→双线对比→真相→释怀/复仇
  
  亡灵视角：死后成为旁观者（能看到活人看不到的）
  双线对比：同一时间点，受害者的平静vs加害者的狂欢
  释怀/复仇：亡灵的选择（不是所有亡灵都要复仇）

【独特技巧】：
  1. 亡灵规则清晰：能做什么、不能做什么（规则要提前说清）
  2. 双线时间同步：同一时间点切换POV，愤怒值拉满
  3. 终极释怀：亡灵选择放下，比复仇更打动读者
```

## 类型EE：都市奇幻甜宠·误会流
**代表作**：我养了一只女魅魔

```
【人设】
- 主角：普通都市人（意外收养奇幻生物）
- 奇幻生物：女魅魔/妖怪/精灵（误会自己是宠物）
- 误会链：奇幻生物用"宠物逻辑"理解人类世界

【虐点节奏】（其实是甜点节奏）
开篇：意外相遇（主角捡到/买到奇幻生物）
第1阶段：误会建立（奇幻生物误以为自己是宠物）
第2阶段：甜宠日常（奇幻生物用宠物逻辑撒娇）
第3阶段：误会升级（奇幻生物"保护"主角→吃醋/破坏约会）
第4阶段：真相揭露（奇幻生物真实身份曝光）
第5阶段：HE收尾（主角接受奇幻身份→甜蜜同居）

【核心公式】：
  意外相遇→误会建立→甜宠日常→误会升级→真相揭露→HE
  
  误会流：奇幻生物用"宠物/妖怪逻辑"理解人类世界
  甜宠：不是虐恋，是全程撒糖
  奇幻设定：不解释太多，保留神秘感和萌点

【独特技巧】：
  1. 误会流：奇幻生物的理解与人类完全不同→笑点+萌点
  2. 甜宠不虐：全程撒糖，没有胃爽文的虐点
  3. 奇幻设定轻量化：不铺大世界观，专注日常甜宠
```
